## Getting started

* Required dependencies
  * pip install -I pyyaml==3.12
  * pip install -I numpy==1.14.0
  * pip install -I protobuf==3.4.0
  * python2 (python3 not supported)

* Limit
  * only caffe model supported
  * only pb/data is file supported
